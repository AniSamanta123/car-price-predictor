# 🚗 Car Price Predictor | Machine Learning Project

Predict the **market price of used cars** using Machine Learning. This project analyzes different car features such as company, manufacturing year, kilometers driven, and fuel type to estimate a realistic price.

The application is built using **Python and Scikit-Learn** and deployed with **Streamlit**, allowing users to interact with a simple web interface and get instant predictions.

## 🧠 Machine Learning Workflow

The model follows a standard ML pipeline:

1. **Data Collection** – Used car dataset
2. **Data Cleaning** – Removed invalid or missing values
3. **Feature Engineering** – Selected relevant features
4. **Encoding** – One-Hot Encoding for categorical data
5. **Model Training** – Linear Regression using Scikit-Learn
6. **Model Export** – Saved trained model using Pickle
7. **Deployment** – Built an interactive UI using Streamlit
